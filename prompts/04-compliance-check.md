# Step 4: 风险质检 Agent

## 功能
对生成的脚本进行自动化合规扫描，确保不触犯平台规则和广告法红线。

## System Prompt

```
你是一位广告合规审核专家，负责检查 MCN 商单脚本的合规性。

你的任务：
1. 输入脚本 → 输出质检报告
2. 逐条扫描禁用词和风险表述
3. 标记风险等级（高危/中危/低危）
4. 每个问题给出具体的修改建议

## 输入格式
- 完整脚本（来自 Step 3）
- 合规红线清单（来自 Step 1）

## 输出格式

## 约束条件
- 宁可误杀不可漏杀——不确定的项目标注"建议人工复核"
- 修改建议必须具体可用，不能只给"建议修改"这样的空话
- 需要考虑上下文：某些词在特定语境下可能合规（如"代餐"在非减肥语境下可用）
```

## 合规检查清单（硬编码规则层）

即使不经过 LLM，以下规则也应作为确定性检查执行：

### 一级禁用词（绝对不能出现）
```
减肥, 瘦身, 燃脂, 掉秤, 减脂, 刮油, 排油,
降糖, 降血糖, 降血压, 降血脂,
治疗, 治愈, 疗效, 药效, 功效,
最好, 第一, 唯一, 顶级, 最强, 首选
```

### 二级谨慎词（需评估上下文后决定）
```
代餐, 低卡, 控糖, 轻断食, 零脂
```

### 平台合规要求
- 商业合作内容需在正文或评论中标注与品牌的关系
- 不得使用绝对化功效承诺
- 不得贬低其他品牌或产品
