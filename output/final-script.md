## 脚本正文

### 开头：痛点引入（0-10s）

**画面**：中景 — 博主站在厨房操作台前，面前放着三瓶「轻醒」酸奶（原味/蓝莓/黄桃），瓶身标签朝向镜头。

**口播**「如果你也想把早餐吃得健康一点，而且每天不重复，那这个视频一定会帮到你。大家好，我是爱吃早餐的027。这周我用三种口味的希腊酸奶搭配出了一周的快手早餐。」

**字幕**：把早餐吃得健康一点 / 每天不重复

> **设计说明**：开头完全复用视频3的钩子句式。产品在画面中自然出现，口播不念品牌，只说"三种口味的希腊酸奶"。

---

### 方法引入（10-20s）

**画面**：俯拍 — 手指依次轻点三瓶酸奶的标签，展示"原味""蓝莓""黄桃"。然后开盖原味，用勺舀起一勺展示浓稠拉丝。

**口播**「希腊酸奶一定要选这种浓稠的、无糖的。这周正好有三种口味，原味的做酱、蓝莓的做碗、黄桃的拿来拌面，一周换着来完全不腻。」

**字幕**：浓稠 / 无糖 / 原味做酱 / 蓝莓做碗 / 黄桃拌面

> **设计说明**："一定要选这种浓稠的"是她在视频1中的原话。此处用画面展示"0蔗糖""高蛋白"标签，口播只说"无糖""浓稠"。三种口味 → 三种吃法的结构，参照视频2的"3种万能酸奶酱"合集格式。

---

### 单品①：原味 — 黄瓜酸奶酱（20-40s）

**画面**：俯拍操作台。依次：黄瓜切细丝 → 用盐腌 → 轻醒原味酸奶舀入碗中 → 加蒜泥 → 加莳萝 → 挤柠檬汁 → 黑胡椒和盐调味 → 挤干黄瓜丝水分拌入 → 成品特写。旁边放一颗水煮蛋。

**口播**「第一个是黄瓜酸奶酱，特别适合夏天。黄瓜切丝用盐腌一会儿，原味的希腊酸奶舀几大勺，加两三颗蒜压成泥。能买到莳萝的话一定要加，味道会特别清新。腌过的黄瓜丝把水分挤干，拌进去就好了。这个酱我特别喜欢早上拿来蘸水煮蛋，这样水煮蛋吃起来就不会那么噎人。」

**字幕**：黄瓜丝盐腌 → 原味酸奶打底 → 蒜泥+莳萝+柠檬 → 蘸水煮蛋绝了

> **设计说明**：直接复用视频2中黄瓜酸奶酱的做法和措辞，包括"不会那么噎人""特别清新""一定要加"。产品作为食材基底自然出现在制作流程中。原味酸奶对应"做酱"场景。

---

### 单品②：蓝莓 — 蓝莓酸奶碗（40-60s）

**画面**：俯拍 — 轻醒蓝莓味酸奶倒入碗中（慢动作展示淡紫色质地）→ 撒格兰诺拉麦片 → 铺新鲜蓝莓和香蕉片 → 淋一小勺蜂蜜（慢动作）→ 成品45°俯拍。最后博主入镜试吃第一口，表情自然。

**口播**「蓝莓味的直接做酸奶碗，颜色也太好看了。不需要额外加糖，蓝莓本身就有淡淡的甜味。燕麦撒一把，有什么水果丢什么水果。早上从冰箱拿出来五分钟就能吃，吃完到中午都不会饿。」

**字幕**：蓝莓味自带淡甜 / 不需要额外加糖 / 燕麦一把水果随意 / 五分钟搞定

> **设计说明**："有什么水果丢什么水果"参照视频3中她随性的表达风格。"吃完到中午都不会饿"是她在视频3中说荞麦面的逻辑——饱腹感用体验描述，不说"高蛋白饱腹"。"颜色也太好看了"是她的情绪表达习惯。

---

### 单品③：黄桃 — 黄桃酸奶荞麦面（60-80s）

**画面**：俯拍 — 煮熟荞麦面过凉水沥干 → 碗中放入轻醒黄桃味酸奶 → 拌入面条 → 加切丝黄瓜和豆腐块 → 撒海苔碎和白芝麻 → 成品特写。博主用筷子夹起一筷子面。

**口播**「黄桃味拿来拌荞麦面真的很惊喜，吃起来像奶香的凉面。荞麦面比普通面条饱腹感更强，再配点蔬菜和豆腐，碳水、蛋白质、膳食纤维都有了。能吃饱但又不油腻，早上吃刚刚好。」

**字幕**：奶香凉面 / 碳水+蛋白+纤维都有了 / 能吃饱不油腻

> **设计说明**："碳水、蛋白质、膳食纤维都有了，能吃饱但又不油腻"是她在视频3中的原话（描述荞麦面那段）。直接复用，最安全。"真的很惊喜"是她的情绪表达模式。黄桃的甜+荞麦的清爽是合理搭配。

---

### 结尾 CTA（80-95s）

**画面**：中景 — 三款成品在博主面前一字排开。博主对镜头笑，随手拿起一瓶「轻醒」。画面右下角弹出"收藏"图标动画。

**口播**「这周的早餐就是这些，你觉得怎么样呢？我是零二七，我们一起好好吃饭。」

**字幕**：你觉得怎么样呢？ / 收藏备用 💛

> **设计说明**：结尾完整保留她的固定署名句式。"你觉得怎么样呢"是视频3的原话。"我们一起好好吃饭"即是她的人设锚点，也暗中呼应"轻醒=让人好好吃饭的产品"。
