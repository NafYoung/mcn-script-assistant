## 拍摄建议

| 事项 | 建议 |
|------|------|
| **总拍摄时长** | 约45-60分钟（含布景+多次拍摄+备用镜头） |
| **光线** | 自然光为主，上午8-10点窗边拍摄效果最佳 |
| **机位** | 手机+三脚架即可，俯拍建议用手机支架或抽拉式支架 |
| **色彩搭配** | 白色酸奶基底 + 蓝莓（紫）+ 黄桃/香蕉（黄）+ 燕麦（棕），对比鲜明 |
| **BGM** | chill beat 类型，音量控制在人声的30-40% |
| **备用镜头** | 建议多拍：开盖特写、拉丝特写、成品45°、成品平视、试吃正面 |
