## 一、内容结构拆解

### 典型视频结构（预估60-90s）

| 时间段 | 结构单元 | 说明 |
|--------|---------|------|
| 0-3s | **钩子** | 画面先出成品特写（高颜值酸奶碗），搭配悬念式开头："别再说早上没时间了，这个5分钟就能搞定" |
| 3-10s | **痛点共鸣** | "以前我也是天天包子吐司，不到十点就饿了" → 建立与打工人受众的连接 |
| 10-40s | **制作过程** | 步骤清晰（3-4步），每步边做边说，节奏快但不乱，关键步骤给特写镜头 |
| 40-55s | **产品/方案亮点** | 自然带出健康替代的优势（高蛋白、饱腹感、快手），不念成分表，用体验说话 |
| 55-70s | **成品展示+试吃** | 高颜值成品各角度展示，试吃镜头（表情自然，不夸张），口播"工作日赶地铁也能吃上这个" |
| 70-85s | **CTA 收尾** | 收藏/点赞提示 + "下一期想看我做什么口味的，评论区告诉我" |

### 钩子模式

- **时间锚定型**："5分钟就能搞定的早餐，比等电梯还快"
- **反差对比型**："以前天天啃包子，现在换这个，一上午不饿"
- **视觉冲击型**：先出成品（打光精致、色彩丰富的酸奶碗），用画面吸引停留

### 口播风格

- **语速**：偏快（适配短视频节奏），咬字清晰
- **语气**：分享型（"我真的太爱这个搭配了"），不教育、不说教
- **人称**：第一人称"我"，拉近与观众距离
- **词汇特征**：口语化，偶尔用"绝了""赶紧试试"等小红书高频词，但不过度

## 三、字幕与包装风格

- **字幕样式**：白色黑边、无衬线字体，关键词（食材名、时间、营养点）用黄色/绿色高亮
- **标题位置**：画面上方1/3处，简短有力（≤15字）
- **BGM**：轻快节奏型（chill beat / lo-fi），音量低于人声
- **特效**：少量，仅用于强调步骤序号或营养亮点（如"蛋白质 15g"弹出动画）

## 四、可复用的内容模式

### 适合「轻醒」酸奶的脚本模式

"快手早餐替代方案" 模式是最佳切入点：
1. 痛点：打工人早餐不健康/饿得快/没时间
2. 方案：用希腊酸奶替代传统主食
3. 呈现：快速制作 + 成品展示 + 试吃
4. 教育：高蛋白=扛饿，0蔗糖=无负担（非减肥话术）
